# 第一节 配置忽略文件

# 1、哪些文件需要忽略？

Eclipse工程特定文件

IDEA工程特定文件

编译产生的二进制文件（对于Maven工程来说就是target目录）

# 2、为什么要忽略这些文件？

与项目的实际功能无关，不参与服务器上部署运行。把它们忽略掉能够屏蔽IDE工具之间的差异，避免这些无关的文件对我们版本控制造成不必要的干扰。
