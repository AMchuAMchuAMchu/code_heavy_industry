# 第四节 SSH方式登录

使用SSH非对称加密技术连接远程库可以实现免密登录

# 1、本地创建SSH密钥文件

#进入当前用户的家目录

$ cd ~

#运行命令生成.ssh密钥目录，注意：这里-C这个参数是大写的C

$ ssh-keygen -t rsa -C atguigu2018ybuq@aliyun.com

#进入.ssh目录查看文件列表

$ cd .ssh

$ ls -lF

#查看id_rsa.pub文件内容

$ cat id_rsa.pub
